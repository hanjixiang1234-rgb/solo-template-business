提交流程

  1. 在这里填完内容，点“打开云端提交页”。
  2. 页面会跳到 GitHub 的预填提交页，你只需要点一次 `Submit new issue`。
  3. 云端开始处理，电脑一上线就会自动同步到本地。

这版入口已经把交互收拢到一个手机页面里了，但最后一步仍然会用 GitHub 帐号提交， 因为云端中枢当前就跑在 GitHub 上。
