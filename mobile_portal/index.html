云端设置

这里优先直连你自己的轻量云后端，不再默认走 GitHub。云端地址和写入口令只会保存在你手机浏览器本地。

如果这两项没填，页面会自动回退到 GitHub 提交。

提交流程

  1. 优先模式：填好后直接提交到你自己的云后端。
  2. 云端会用 OpenAI 处理内容，再把结构化结果留在云端队列里。
  3. 电脑一上线，本地同步脚本会把结果拉回 minder 和 bilibili-cat-meme。

如果你暂时还没把新云后端部署好，页面会回退到 GitHub 预填提交页，不会直接失效。
